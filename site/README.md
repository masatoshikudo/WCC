# WCC 事業サイト（Next.js）

`DESIGN.md` のデザインシステムと `reference/STACK.md` の MVP 手順に基づく実装です。

## 開発

```bash
npm install
npm run dev
```

- [http://localhost:3000](http://localhost:3000) — `/`, `/packages`, `/portfolio`, `/contact`, `/book`

## ドキュメント

- `../reference/implementation_phase5.md` — フェーズ5で置いたファイル一覧
- `../DESIGN.md` — トークン・コンポーネント規約

## ビルド

```bash
npm run build
```
