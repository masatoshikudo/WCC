# ウェディング・コンテンツ・クリエイター（WCC）事業 立ち上げ計画書

本計画書は、経営者1名、WCC（ウェディング・コンテンツ・クリエイター）2名の計3名体制で、日本国内においてWCC事業を立ち上げるためのロードマップおよび検討項目を整理したものです。

---

## 1. 日本市場の現状整理（前提条件）

日本国内でWCC事業を展開するにあたり、以下の市場環境と障壁を前提としてビジネスを設計する必要があります。

### 市場の機会（オポチュニティ）

- **Z世代・ミレニアル世代のSNS需要**：結婚式当日のリアルな様子を即日SNS（Instagram Reels / TikTok）でシェアしたいという「プレ花嫁」のニーズが急増しています[1]。
- **市場の空白地帯**：国内のWCC市場は黎明期であり、プロのクオリティで縦型ショート動画を即日納品できる専門事業者はまだ少なく、先行者利益を獲得しやすい状況です。
- **高利益率モデル**：機材投資が少なく（最新のiPhoneとジンバル等）、原価がほとんどかからないため、非常に高い粗利率を確保できます。

### 最大の障壁（ボトルネック）

- **式場の「持ち込み制限」**：日本の結婚式場の多くは、提携業者以外のカメラマンの持ち込みを禁止、または高額な持ち込み料（3〜10万円）を設定しています[2] [3]。
- **対策の方向性**：
  1. **ゲスト（友人）として参列する「ステルス型」**（※ただしトラブルのリスクあり）
  2. **持ち込み自由な式場・プロデュース会社との提携（BtoBtoC）**
  3. **前撮り・二次会・1.5次会など規制の緩い領域からの参入**

## 2. 検討項目（洗い出し）

事業立ち上げに向けて、3名チームで以下の項目を決定・準備する必要があります。

### A. サービス設計・価格設定

- **パッケージ内容の決定**：
  - 稼働時間（例：挙式のみ4時間、披露宴まで8時間、お支度から二次会まで12時間）
  - 納品物（例：無編集データ全量＋編集済みショート動画3本）
  - 納品スピード（例：**RAWは24時間以内**、編集動画は**別SLA**（例：72時間以内）として明記 — 海外主要プレイヤーと同様に「何がいつ届くか」をパッケージ表で固定する）
  - **パッケージ表**（時間帯・RAWの有無・編集本数・納品時刻の目安）を1枚にまとめ、期待値のズレを防ぐ（`WCC_competitor_comparison_report.md` の示唆）
- **価格設定（プライシング）**：
  - 国内の個人フリーランス相場（3〜5万円）と、海外エージェンシー相場（10〜30万円）のどこにポジショニングするか。
  - 初期はモニター価格（実績作り）で展開し、段階的に値上げする戦略が推奨されます。

### B. ターゲット層と集客チャネル

- **メインターゲット**：
  - 20代半ば〜30代前半のSNSリテラシーが高いプレ花嫁。
  - 特に、持ち込み自由なオリジナルウェディング（TRUNK BY SHOTO GALLERYやCRAZY WEDDINGなど）を検討している層。
- **集客チャネル**：
  - **Instagram / TikTok（最重要）**：自社の公式アカウントで、実際の納品イメージ（エモい縦型動画）を継続的に発信。
  - **プレ花嫁コミュニティ**：X（旧Twitter）やInstagramの「#プレ花嫁」「#2025秋婚」などのハッシュタグを活用したアウトバウンド・アプローチ。

### C. チーム内の役割分担

- **経営者（1名）**：
  - 事業戦略、マーケティング（SNS運用・広告）、顧客対応（問い合わせ〜契約）、提携先開拓（BtoB営業）、財務管理。
- **WCC（2名）**：
  - 現場での撮影ディレクション、新郎新婦・ゲストとのコミュニケーション、撮影データの即日編集、納品。
  - ※2名体制であるため、1日に最大2件の稼働が可能、あるいは1件に対して2名体制で多角的なアングルをカバーするプレミアムプランの提供が可能です。

### D. 法務・契約・リスク管理

- **利用規約の作成**：キャンセルポリシー、式場側の規制により撮影が制限された場合の免責事項、機材トラブル時の対応。
- **著作権・肖像権の取り扱い**：撮影した素材を自社のSNSやポートフォリオで実績として公開するための同意書（許諾）の取得。

## 3. 事業立ち上げロードマップ（今後6ヶ月の計画）

以下の3フェーズで事業を立ち上げ、軌道に乗せます。海外主要WCC・市場調査との対応関係は `WCC_competitor_comparison_report.md` および `reference/ウェディング・コンテンツ・クリエイター（WCC）市場動向・ビジネスモデル徹底調査レポート.md` を参照する。

### フェーズ1：準備・テストマーケティング（Month 1-2）

**目標：サービスパッケージの確定と、ポートフォリオ（実績）の構築**

1. **機材調達とテスト撮影**
   - 最新のiPhone（Proモデル推奨）、ジンバル、モバイルバッテリー、大容量クラウドストレージの準備。
   - **標準機材セットを文書化**（機種、予備電源、納品用クラウドの運用方針）。海外では機材を型番・スマホ世代まで明記する例がある（The Wedding Content Co 等）。
   - テスト撮影では **縦型（9:16）・低照度・動きのあるシーン** をチェックリスト化。2名体制の場合は **角度・役割分担**（プレミアムプラン用）を試す。

2. **モニター撮影の実施**
   - 知人の結婚式や、SNSで募集した「無料または格安モニター（ポートフォリオ掲載許可が条件）」を **3〜5件** 実施。
   - モニター段階から **稼働時間・RAW納品の有無・編集本数・納品目安時刻** を仮パッケージで固定し、**事後アンケートで期待値とズレを記録**する。
   - 可能なら **前撮り・二次会・1.5次会** など、持ち込みと相性の良いシーンを混ぜ、式場リスクの学習を早める（セクション1の対策2・3と連動）。

3. **SNSアカウントの立ち上げ**
   - モニター撮影で得た素材を編集し、InstagramとTikTokに投稿開始。世界観（トーン＆マナー）を統一する。
   - 投稿は次の **3本柱** で設計する：（1）**納品イメージ**（縦型ショート）（2）**撮影BTS・機材**（3）**啓蒙**（「WCCとは」「プロフォト・ムービーとの違い」）。市場では「アンチ・パーフェクション」「その場の空気のシェア」が価値として説明されている。
   - ハッシュタグは `#プレ花嫁` 等に加え、**英語タグ（例: #weddingcontentcreator）** も試行しリーチを検証。

4. **契約書・規約の整備**
   - 利用規約や同意書のフォーマットを作成。
   - 契約に必ず盛り込む論点：**(a)** 納品の定義（RAWの目安本数・編集本数・遅延時の扱い）**(b)** 式場規約等により撮影が制限・禁止された場合の免責 **(c)** ポートフォリオ・SNS掲載の肖像・著作権 **(d)** ゲスト映り込みとSNS公開の範囲。
   - **支払い条件**は海外事例に倣い **デポジット（例：50%）＋残金タイミング** を検討する（Content For Brides、The Wedding Content Co 等の型）。

5. **（フェーズ1内の成果物）**
   - **パッケージ表ドラフト**（時間帯×納品物×価格帯のたたき台）。海外の「表で比較できる」サイト（Shutter and Sound、The Social Sisters 等）を参考に1枚に整理。
   - **「WCCとは」内部原稿**（LP・SNS共通の説明コピー用）。海外では「What is a wedding content creator?」の教育セクションを持つ例がある（Wedding Content Creator UK 等）。

### フェーズ2：集客開始・アーリーアダプター獲得（Month 3-4）

**目標：SNS経由での自社集客（月間3〜5件の受注）**

1. **パッケージの正式リリース**
   - 初期は相場よりやや安めの価格設定（例：**5〜8万円**をライト〜ミドル帯）でスタートし、実績と口コミを蓄積する。
   - **3段階パッケージ**（例：4〜6時間／8〜10時間／2名体制プレミアム）を想定。海外相場ではライト $500〜$900帯、スタンダード $1,000〜$1,500 帯などのパッケージ化が一般的（市場調査レポート）。
   - **オプション**：時間延長、編集本数追加、前日リハーサル同行（海外で価値が出る場面あり）。証言ベースでは「もっと長く予約すればよかった」というインサイトがあり、**延長をメニュー表で視覚化**する。
   - **MVP Web**：`STACK.md` の Step 1〜3 に沿い、**LP・料金・実績・問い合わせ**に加え、**FAQ**（フォトとの違い、式場・他ベンダーへの事前連絡、持ち込み）を掲載。比較レポートでは「HOME・サービス／料金・実績・問い合わせ・FAQ」が多数社で共通。

2. **SNS運用の強化**
   - プレ花嫁が検索するハッシュタグへの露出強化。撮影の裏側（BTS）や、「なぜWCCが必要か」の啓蒙コンテンツを発信。
   - **月に1本は啓蒙特化**（カテゴリ教育）。日本市場では「WCC」より **「スマホ縦型・当日納品」** など併記し理解を補助する（市場調査レポートのローカライズ示唆）。
   - **note や Instagram ガイド** など、検索流入用の軽い教育コンテンツを検討（個人ブランド型の海外事例の要素）。

3. **顧客対応フローの確立**
   - 問い合わせ〜オンライン面談〜契約〜事前ヒアリング〜当日の動線確認までのオペレーションを最適化。
   - **スクリーニング**：初回で **式場名・持ち込み可否・フォト・プランナー連絡先** を必須項目化（トラブル予防）。
   - **CRM上の段階**：問い合わせ → 面談 → 見積 → 契約 → **式の数週間前の最終打合せ**（編集の好み・音楽・NG人物・他サプライヤー連携）。海外では事前コンサル・直前フォームの二段構えの例がある。

4. **（フェーズ2の種まき）**
   - **BtoB向け一文**（プランナー・フォト向け「当日BTS撮影の外部パートナー」）と **1枚PDF** を用意し、フェーズ3の本格提携に接続（Wed Social、The Wedding Content Co 型の二系統の簡易版）。

### フェーズ1〜2で取り入れる海外ベストプラクティス（要約）

| 領域 | 取り入れる内容 |
| :--- | :--- |
| プロダクト | **24時間以内にRAW**、編集は別SLAで明記。稼働時間・編集本数を表で固定。 |
| 価格 | デポジット＋残金、**時間延長オプション**の明示。 |
| マーケ | **動画ポートフォリオ＋顧客の声＋啓蒙**の三本柱。 |
| オペレーション | 他ベンダー・式場への**事前連絡**をテンプレ化。 |
| リスク | FAQ・契約で **持ち込み制限** を正面から説明（日本固有の安心材料）。 |

### フェーズ3：単価向上・BtoB提携の開拓（Month 5-6）

**目標：事業の黒字化、適正価格への移行、提携先の確保**

1. **価格改定**：実績と需要の増加に伴い、本来の適正価格（例：10〜15万円）へ段階的に値上げ。
2. **BtoB営業の開始（経営者の主務）**：持ち込みに寛容なプロデュース会社、フリーランスのウェディングプランナー、既存のフォトグラファーに対して「WCCオプション」としての提携営業を行う。
3. **オプションの拡充**：前撮り同行プラン、二次会特化プラン、SNS運用代行プランなどのアップセル商材を開発。

## 4. 成功のためのキーファクター

- **式場スタッフとの協調性**：現場でプロのカメラマンや式場スタッフの邪魔にならないよう、極めて高いコミュニケーション能力と「黒子に徹する」姿勢が求められます。
- **圧倒的な納品スピード**：WCCの最大の価値は「熱量が高いうちにSNSでシェアできること」です。24時間以内の納品は絶対に死守するオペレーションを構築してください。
- **「エモさ」の言語化と再現性**：ターゲットが求める「少しブレた生々しさ」「トレンドの音源に合わせたテンポ感」を、WCC2名の間で属人化させず、チームとしてのクオリティ基準を設けることが重要です。

## 5. KPI・計測（フェーズ2以降）

月間 **3〜5件** の受注目標を支えるため、次を最低限トラッキングする。

| 指標 | 内容 |
| :--- | :--- |
| リード数 | 問い合わせ・DM・紹介の件数 |
| 成約率 | 見積提出〜契約の転換率 |
| 単価 | パッケージ別・オプション込みの平均 |
| オペレーション | **納品遅延ゼロ**の日数、クレーム・修正回数 |
| 紹介・リピート | 紹介経由の割合 |
| 期待値管理 | 成約時に「プロのムービー・アルバムとの違い」の**理解確認**（チェックまたは署名）— 期待値ズレによる不満を抑える（比較レポートの論点） |
