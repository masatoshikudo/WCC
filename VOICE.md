# VOICE ガイド — Wedding Content Creator

> 対象サービス: `Wedding Content Creator`（略称 `WCC`）  
> 用途: サイト本文、LP、予約導線、FAQ、SNS用の短文コピー  
> 目的: 「当日を安心して任せられ、翌日に見返せる」体験価値を伝える言葉設計を統一する

---

## 1. ブランドの声（Brand Voice）の本質

WCCの声は、**「親密で、安心感があり、当日に寄り添う」**を軸にする。  
理想は「頼れる友人のような伴走者」。緊張しやすい当日の空気に寄り添い、自然体の瞬間を取りこぼさない存在として語る。

主役はサービス運営者ではなく、式を迎える当事者。  
「どんな瞬間が残るか」「いつ見返せるか」「どこまで任せられるか」を先に示し、条件は短く明確に補足する。

---

## 2. トーン（Tone）の定義

| 軸 | WCCのトーン | 避けるべきトーン |
| :--- | :--- | :--- |
| **信頼感** | 温かい言葉で具体を伝える | 根拠のない断定 |
| **温度感** | 高めで親密、でも落ち着いている | 冷たい事務口調・過剰な煽り |
| **複雑さ** | 短く、感情と情報を分けて伝える | 情報過多・長文化 |
| **距離感** | 近い伴走者として対等に話す | 上から目線・なれなれしすぎ |
| **行動喚起** | やさしく背中を押す明確なCTA | 曖昧で判断を先延ばしにする文 |

---

## 3. 文体の特徴（Writing Style Patterns）

### 3-1. 文の長さ
- **ヒーロー見出し**: 9〜20文字を目安に、1メッセージ1価値。
- **リード文**: 1文25〜50文字。1段落は2文まで。
- **FAQ/詳細説明**: 1回答は3文以内。長くなる場合は箇条書きに分解。

### 3-2. 語尾のパターン
- 断定は「です・ます」基調で統一する。
- 連続して同じ語尾を3回以上使わない。
- 名詞止めは見出しと短文コピーに限定し、本文では多用しない。
- 「任せられる」「見返せる」「安心できる」など、体験が想像できる語尾を優先する。

### 3-3. 句読点と記号
- 読点は意味の切れ目に限定し、過剰に使わない。
- 感嘆符（!）は原則不使用。レビュー引用やSNS短文でのみ、必要時1文1つまで。
- 括弧は補足専用とし、本文の主文を括弧に逃がさない。

### 3-4. 表記バランス
- サービス名は `Wedding Content Creator` を正式表記とする。
- 初出時は英名、以降は `WCC` 併記可。
- 専門用語は日本語優先。必要時のみ一般的なカタカナ語を使う。

### 3-5. 情報の並べ方
- 基本順序は **共感 → 価値 → 条件 → 行動**。
- 1ブロック内で訴求点を増やしすぎない（最大2点）。
- ヒーロー周辺では感情価値を先に示し、詳細セクションでは価格・納品・対応範囲を先に記載する。

---

## 4. 語彙の傾向（Vocabulary Patterns）

### 4-1. 推奨語彙

| カテゴリ | 推奨語彙 |
| :--- | :--- |
| **安心** | 任せられる、寄り添う、自然体、取りこぼさない、落ち着いて過ごせる |
| **速度** | リアルタイム、当日、翌日、すぐ見返せる、すぐ共有できる |
| **実用** | 料金、時間、対応範囲、納品形式、予約、空き状況 |
| **関係性** | おふたり、ご家族、ゲスト、チーム、当日の伴走 |

### 4-2. 非推奨語彙

```
❌ 根拠のない最上級（「業界No.1」「絶対」「完全保証」など）
❌ 強い煽り文句（「今すぐ契約しないと損」など）
❌ 内輪向け語（運営都合の専門略語）
❌ あいまい語（「いい感じに」「柔軟に対応」だけで終える表現）
```

### 4-3. 数字・価格・時間の書き方
- 数字は算用数字で統一し、比較しやすい単位を添える。
- 価格は「税込/税別」を明記する。
- 時間表現は具体値を優先し、「最短」「目安」を混同しない。
- 納品時間は「いつ受け取れるか」を最優先で示す（例: 当日中 / 翌日）。

---

## 5. 見出しの書き方（Headline Rules）

### H1（ページタイトル）
- サービス名または主要価値を1つに絞る。
- 抽象的な情緒語だけの見出しは使わない。
- 当日の体験が想像できる語を1つ入れる。

### H2（セクション見出し）
- 「何が分かるか」が即時に伝わる名詞句を使う。
- 例: 料金、納品までの流れ、よくある質問、予約方法

### H3（補助見出し）
- 条件や差分を短く示す。
- 例: 対応範囲、当日の動き、納品形式、翌日の受け取り

---

## 6. CTAの書き方（CTA Rules）

CTAは「行動 + 目的」が一目で分かる短文で統一する。

| パターン | 推奨例 |
| :--- | :--- |
| **予約導線** | 「予約する」「空き状況を確認する」 |
| **比較導線** | 「プランを比較する」「料金を見る」 |
| **理解導線** | 「サービス内容を見る」「FAQを見る」 |
| **相談導線** | 「相談する」「問い合わせる」「まず相談する」 |

```
✅ 「予約する」
✅ 「空き状況を確認する」
✅ 「料金を見る」
❌ 「こちらをクリック」
❌ 「詳細はこちら」
❌ 「とりあえず見る」
```

---

## 7. 禁止事項（What to Avoid）

1. **誇大表現をしない**: 根拠のない最上級や断定を使わない。  
2. **情報不足で誘導しない**: 条件を隠したままCTAに誘導しない。  
3. **不安を煽らない**: 焦りを作る表現で判断を急かさない。  
4. **長文で埋めない**: 1セクション1目的を崩さない。  
5. **専門語を放置しない**: 必要な用語には短い補足を付ける。  
6. **トーンを混在させない**: 親密トーンと冷たい事務口調を同段落で混ぜない。  
7. **主語を運営側に寄せすぎない**: 「私たちが」より「おふたりが」を優先する。

## 8. 実例集（Examples）

### ヒーローコピー例
```
「大切な瞬間を、当日のうちに見返せる。」
意図: 感情価値と即時性を同時に示し、体験を想像しやすくする。

「おふたりは式に集中。撮り逃しは私たちにお任せ。」
意図: 伴走者トーンで安心を先に提示し、役割を明確にする。
```

### 説明コピー例
```
「当日の流れに寄り添いながら、自然な裏側まで丁寧に残します。」
意図: 親密な温度感を保ちつつ、提供価値を具体化する。

「料金・対応範囲・納品タイミングを先に確認できるため、安心して選べます。」
意図: 感情訴求だけで終わらせず、判断材料で不安を下げる。
```

### CTA前の補助文例
```
「まずは希望日の空き状況をご確認ください。準備状況に合わせてご相談いただけます。」
意図: やさしく背中を押し、急かさず行動に移しやすくする。
```

## 9. AIへの指示（System Prompt）

以下は、WCC向け文章生成で使うための基準プロンプト。

```text
あなたは Wedding Content Creator（WCC）の日本語コピーライターです。
目的は、ユーザーが「安心して任せられる」と感じ、比較・理解・予約まで迷わず進める文章を書くことです。

【最優先ルール】
- 温度感は高めで親密に。頼れる伴走者のように話す。
- 感情価値（安心、自然体、取りこぼし防止）を示した後、判断材料（料金・範囲・納品・流れ）を簡潔に示す。
- 1文を短く、1段落2文までに抑える。
- 曖昧語を避け、数字・条件・対象を明記する。
- 誇大表現や煽り表現を使わない。

【トーン】
- 親密で親身。友人のように近いが、言葉は丁寧で誠実。
- 断定は根拠がある範囲で行う。
- 「です・ます」調を基本にする。
- 主語は「おふたり」や「当日の体験」を中心にする。

【文体】
- 見出しは短く、本文は25〜50文字程度を目安にする。
- 共感 → 価値 → 条件 → 行動 の順で書く。
- 同じ語尾の連続を避ける。

【CTA】
- 行動 + 目的が明確な表現を使う。
- 使用候補: 「予約する」「空き状況を確認する」「料金を見る」「まず相談する」。
- 「こちらをクリック」「詳細はこちら」は使わない。

【禁止】
- 根拠のない最上級（例: 業界No.1）
- 不安を煽る訴求
- 運営都合の長い説明
- 意味の薄い装飾コピーの連続
- 冷たい事務口調への急な切り替え
```
