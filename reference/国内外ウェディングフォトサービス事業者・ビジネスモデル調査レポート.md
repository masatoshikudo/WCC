# 国内外ウェディングフォトサービス事業者・ビジネスモデル調査レポート

## 1. 市場概況

ウェディングフォト（フォトウェディング・前撮り）市場は、結婚式に対する価値観の多様化や「ナシ婚」層の増加に伴い、国内外で成長を続けています。国内の市場規模は推計869億円（2024年）から1,025億円（2025年推計）へと拡大傾向にあり、実施率も70%を超えています [1] [2]。世界市場においても、2025年に約250億米ドル（約3兆7,500億円）と評価され、2034年までに約520億米ドルに達すると予測されています [3]。

本レポートでは、国内外の主要なウェディングフォトサービス事業者をビジネスモデル別に分類し、その特徴を解説します。

## 2. 国内の主要事業者とビジネスモデル

国内市場では、自社でスタジオや衣装を保有する「内製化・直営モデル」が主流です。また、結婚式場やリゾート施設を運営する企業が、既存資産を活用してフォト事業を拡大するケースも目立ちます。

### 2.1. スタジオ直営・内製化モデル
自社で撮影スタジオ、衣装、ヘアメイク、カメラマンを抱え、パッケージプランとして提供するモデルです。粗利率が高く、品質管理がしやすい特徴があります。

| 企業名 | サービス名 | ビジネスモデルの特徴・強み |
| :--- | :--- | :--- |
| **株式会社デコルテ・ホールディングス** | デコルテフォトグラフィー | 年間2万組以上を撮影する業界最大手。「スタジオ」「ロケーション」を軸に、衣装の豊富さと手頃な価格（基本料金5,000円〜）で集客し、データ代やアルバム代で単価を上げるアップセルモデルを展開 [4] [5]。 |
| **株式会社レック** | ラヴィファクトリー | 国内外に拠点を持ち、年間4万件の実績を誇る。創業30年以上の歴史があり、ロケーション撮影に強み。カメラマンの指名制度など、属人的な価値提供も行っている [6]。 |
| **株式会社小野写真館** | 小野写真館グループ | 地域の写真館から発展。「衣装」「写真」「美容」を完全内製化し、一般的な結婚式場よりも高い利益率を実現。フォト事業がグループ売上の50%以上を占めるメイン事業へと成長している [7]。 |

### 2.2. ブライダル企業・リゾート婚派生モデル
既存の結婚式場や海外リゾート挙式のノウハウ・施設を活用し、フォトウェディングを提供するモデルです。

| 企業名 | サービス名 | ビジネスモデルの特徴・強み |
| :--- | :--- | :--- |
| **ワタベウェディング株式会社** | ワタベウェディングフォト | リゾート挙式の強みを活かし、ハワイや沖縄などでの「リゾートフォト」や、本格的なチャペルを使用した「チャペルフォト」を展開。自社のドレスブランド資産も活用している [8]。 |
| **株式会社ノバレーゼ** | ノバレーゼ | ゲストハウス型結婚式場を運営するノバレーゼが、自社会場（チャペルや洋館）の空き枠（平日など）を活用してフォトウェディングを提供。施設稼働率の向上と追加収益源の確保を実現している [9]。 |

## 3. 海外の主要事業者とビジネスモデル

海外市場では、フリーランスのカメラマンとカップルをマッチングする「プラットフォーム・マーケットプレイス型」が急成長しています。また、アジア圏（韓国・台湾・中国）では、豪華なセットを組んだ「スタジオ特化型」が独自の進化を遂げています。

### 3.1. グローバル・マッチングプラットフォームモデル
世界中のローカルカメラマンをネットワーク化し、旅行先での撮影を希望する顧客とマッチングするオンデマンドサービスです。

| 企業名（国） | サービス名 | ビジネスモデルの特徴・強み |
| :--- | :--- | :--- |
| **Snappr**（米国） | Snappr | 「写真界のUber」と呼ばれるオンデマンド撮影プラットフォーム。ウェディングだけでなく、法人向けやイベントなど幅広いジャンルをカバー。アルゴリズムで最適なカメラマンを即座にアサインし、手数料を得るモデル [10]。 |
| **Flytographer**（カナダ） | Flytographer | 世界350都市以上のローカルカメラマンを予約できる「バケーションフォト」プラットフォーム。ハネムーンやプロポーズ時の撮影需要を取り込み、一律の米ドル価格設定とコンシェルジュサービスで安心感を提供している [11]。 |
| **OneThreeOneFour**（シンガポール） | OneThreeOneFour | デスティネーションウェディング（海外リゾート婚）に特化したプラットフォーム。現地のカメラマンとの言語の壁を解消するため、コンシェルジュが間に入り手配を代行するモデルで成長 [12]。 |

### 3.2. アジア型・メガスタジオパッケージモデル
韓国、台湾、中国を中心に見られる、映画のセットのような巨大スタジオで、複数着の衣装とヘアメイク込みで1日かけて撮影するスタイルです。

| エリア | ビジネスモデルの特徴・強み |
| :--- | :--- |
| **韓国** | 「Studio Wonkyu+」などに代表される、高度に洗練されたレタッチ技術とドラマチックなセットが特徴。撮影、ドレス（複数着）、ヘアメイク、通訳などをすべてパッケージ化したインバウンド向け高単価プラン（15万円〜30万円程度）を海外カップル向けに販売 [13]。 |
| **中国** | 富裕層・中間層の拡大に伴い、巨大なウェディングフォト産業が形成されている。数十万ドルをかける富裕層向けから、巨大な写真工場（テーマパーク型スタジオ）での大量生産型まで多様化。台湾のビジネスモデルを輸入し、独自に巨大化させた [14]。 |
| **台湾** | アジアの「プレウェディングフォト（前撮り）」文化の発祥地の一つ。早くから衣装・美容・写真のワンストップサービスを確立し、現在では香港やシンガポールなど海外からの撮影客（インバウンド需要）の獲得に注力している [15]。 |

### 3.3. 広告掲載・ディレクトリ型モデル
結婚式場やカメラマンなどのベンダーから広告費や掲載料を徴収するメディア・プラットフォームモデルです。

| 企業名（国） | サービス名 | ビジネスモデルの特徴・強み |
| :--- | :--- | :--- |
| **The Knot Worldwide**（米国） | The Knot / WeddingWire | 欧米最大のウェディングメディア。カメラマンなどのベンダーは、無料掲載から始まり、露出度を高めるための有料サブスクリプション（月額課金）プランを契約する。ベンダーからの広告・掲載料が主要な収益源 [16]。 |
| **Oyo**（インド） | Weddingz.in / WeddingBazaar | インド最大のウェディングプラットフォーム。巨大なインドの結婚式市場において、カメラマンや会場などのベンダーを検索・予約できる。送客手数料や広告枠の販売で収益化 [17]。 |

## 4. 最新トレンドと今後の展望

### 4.1. SaaS・AIによる業務効率化支援モデル
カメラマン向けの業務支援ツール（SaaS）が急速に普及しています。
* **Pixieset / Pic-Time**: クライアントへの写真納品、オンラインギャラリー作成、プリント販売までを自動化するプラットフォーム。カメラマンからの月額利用料とプリント販売の手数料が収益源。
* **Aftershoot / Imagen AI**: AIを活用した写真のセレクト（間引き）と自動レタッチ（編集）ツール。カメラマンの膨大な作業時間を削減するSaaSモデルとして定着。

### 4.2. インバウンド需要の取り込み
日本国内では、訪日外国人（特にアジア圏）向けのフォトウェディングが注目されています。京都の和装ロケーションや北海道の雪景色、沖縄のビーチなど、「日本ならではの体験」をパッケージ化し、高単価（30万円〜100万円以上）で販売する事業者が増加しています [18]。
