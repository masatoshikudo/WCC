## 横断テーマ

### SNS × LP の連動

SNS は認知獲得の主戦場、LP は信頼担保の場（検討確認の場）。
- SNS で作例を見せる → LP で信頼を確認する → 問い合わせへ転換
- 作例のない状態では LP のコンバージョンは期待できない
- SNS アカウント立ち上げ（M1）が全てのボトルネック

### 世界観の一貫性

> **「翌朝、二人で昨日を笑い合うための動画」**（`docs/WCC_PRODUCT.md` 1.1）

全フェーズを通じて、この体験定義を中心に判断する。
機能追加・価格改定・集客施策を検討する際は必ずここに立ち返る。

### 未確定事項（実撮影で確定）

- 動画 2 本の尺と棲み分け（短尺 SNS 向け + 中尺家族共有向けの構成想定）
- 全素材の渡し方（未編集クリップ全部か選別済みか）
- 編集スタイルの言語化（実案件を通じて型を作る）

### 未解決の課題（フェーズ横断）

- 本番 admin 画面のパスワード問題（ローカルと Vercel で値が異なる可能性 — 要調査）
