# 国内外ウェディングフォト・ムービーサービス事業者・ビジネスモデル調査レポート

## 1. 市場概況

ウェディングフォト（フォトウェディング・前撮り）およびウェディングムービー市場は、結婚式に対する価値観の多様化や「ナシ婚」層の増加に伴い、国内外で成長を続けています。国内のウェディングフォト市場規模は推計869億円（2024年）から1,025億円（2025年推計）へと拡大傾向にあり、実施率も70%を超えています [1] [2]。世界市場においても、ウェディング写真・映像サービス市場は2025年に約3,566億米ドルと評価され、2034年までに約9,778億米ドルに達すると予測される巨大な産業です [3]。

特に近年は、写真だけでなく「映像（ムービー）」に対する需要が高まっています。国内では披露宴で上映する「オープニングムービー」「プロフィールムービー」「撮って出しエンドロール」が定番化しており、海外ではシネマティックな「ウェディングフィルム」の制作が富裕層を中心に人気を集めています。

本レポートでは、国内外の主要なウェディングフォト・ムービーサービス事業者をビジネスモデル別に分類し、特に**「フォトとムービーの両方を展開している企業」**に焦点を当てて解説します。

## 2. 国内の主要事業者とビジネスモデル

国内市場では、自社でスタジオや衣装を保有する「内製化・直営モデル」や、既存の結婚式場アセットを活用するモデルが主流です。また、ムービー専門の制作会社がフォト事業へ参入するケースや、逆にフォトスタジオが動画サービスを拡充する動きが活発化しています。

### 2.1. フォト・ムービー両方展開モデル（スタジオ直営・内製化）
自社で撮影スタジオ、衣装、ヘアメイク、カメラマン（スチール・ビデオ）を抱え、パッケージプランとして提供するモデルです。フォトとムービーをセット販売することで顧客単価（LTV）を最大化しています。

| :--- | :--- | :--- |
| **株式会社デコルテ・ホールディングス** | デコルテフォトグラフィー | 年間2万組以上を撮影する業界最大手。「ウェディングフォトムービー」として、プロの写真・動画と顧客のセルフ動画を組み合わせた3分間のムービーや、SNS用ショート動画（35秒）を提供。フォト撮影のついでに動画も残せる手軽さでクロスセルを図る [4] [5]。 |
| **株式会社レック** | ラヴィファクトリー | 国内外に拠点を持ち、年間4万件の実績を誇る。ロケーション撮影に強みを持ち、「前撮りムービー」プラン（1時間〜5時間）を展開。フォトグラファーとビデオグラファーの2名体制で、シネマティックな映像と写真を同時に残せる高単価プランが特徴 [6] [7]。 |

## 3. 海外の主要事業者とビジネスモデル

海外市場では、フリーランスのビデオグラファーとカップルをマッチングする「プラットフォーム型」や、顧客自身が撮影した動画を編集する「クラウドソーシング型」など、多様なビジネスモデルが存在します。

### 3.1. グローバル・マッチングプラットフォームモデル
世界中のローカルカメラマン（スチール・ビデオ）をネットワーク化し、顧客とマッチングするオンデマンドサービスです。

| 企業名（国） | サービス名 | ビジネスモデルの特徴・強み |
| :--- | :--- | :--- |
| **Snappr**（米国） | Snappr | 「写真界のUber」と呼ばれるプラットフォーム。フォトグラファーだけでなくビデオグラファーの手配も可能。ウェディングから企業向け（レストラン、不動産など）まで幅広くカバーし、API連携によるエンタープライズ向け事業が収益の柱に成長 [14] [15]。 |
| **Flytographer**（カナダ） | Flytographer | 世界350都市以上のローカルフォトグラファーを予約できるプラットフォーム。主にバケーションやプロポーズの「写真」に特化しており、本格的なウェディングやムービー撮影は対象外（特別リクエストによる紹介のみ）としている点が特徴 [16]。 |
| **OneThreeOneFour**（シンガポール） | OneThreeOneFour | デスティネーションウェディングに特化。現地のフォトグラファー・ビデオグラファーの手配、衣装、ヘアメイク、移動手段などをコンシェルジュがフルパッケージで代行。アジア圏の富裕層を中心に支持を集める [17]。 |

### 3.2. クラウドソーシング・動画編集特化モデル
プロのカメラマンを派遣せず、ゲストが撮影した素材を活用するモデルや、編集作業のみを請け負うBtoBモデルです。

| 企業名（国） | サービス名 | ビジネスモデルの特徴・強み |
| :--- | :--- | :--- |
| **Storymix Media**（米国） | WeddingMix | ゲストや新郎新婦がスマホや専用のレンタルカメラで撮影した動画・写真のデータを集め、プロが編集して一つのウェディングムービーに仕上げる「クラウドソーシング型」モデル。プロを雇うより安価に、かつゲスト目線のリアルな映像を残せるのが強み [18]。 |
| **Ours Global / Nobacklog** | Wedding Video Editing Outsourcing | 撮影は現地のビデオグラファーが行い、「編集作業」のみをインドやフィリピンなどのオフショア拠点に外注するBtoBモデル。現地の制作会社は編集の手間を省き、撮影件数を増やすことで利益率を向上させている [19] [20]。 |

### 3.3. 広告掲載・ディレクトリ・SaaS統合型モデル
ベンダー（ビデオグラファー含む）から広告費を徴収するメディアモデルと、結婚式準備のSaaS機能を提供するモデルです。

## 4. 最新トレンドと今後の展望

### 4.1. 「フォト＋ショートムービー」のセット化
TikTokやInstagramリールなど、縦型ショート動画の普及により、長尺の記録映像よりも「SNSでシェアしやすい数十秒〜数分のショートムービー」の需要が急増しています。デコルテフォトグラフィーのように、写真撮影のついでに手軽に動画を残せるプランが、今後のスタンダードになると予想されます。

### 4.2. 映像制作会社のフォト事業参入とハイブリッド化
PICNIKOやシラフジのように、元々映像制作に強みを持つ企業が「写真も撮れる」ことを武器にフォトウェディング市場に参入するケースが増えています。顧客側も「写真と動画で別々の業者を探す手間」を省けるため、ワンストップで両方を提供できるクリエイティブチームの価値が高まっています。

### 4.3. AIによる動画編集の自動化
写真領域（Aftershootなど）で進んでいるAI化の波が、動画領域にも波及しています。数時間分の映像素材からハイライトシーンを自動抽出し、音楽に合わせて自動編集するAIツールの導入が進めば、ウェディングムービーの制作コストと納期は劇的に圧縮され、より安価なサービスの登場や利益率の向上が見込まれます。
