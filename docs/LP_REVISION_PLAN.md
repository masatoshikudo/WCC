# LP Revision Plan

既存 LP (`for-your-wedding-day.com`) の改修計画。
`docs/WCC_PRODUCT.md` のメイン体験・世界観・集客戦略に合わせたコピー・構成の調整。

> ステータス: Phase 1 実装中
> 関連ドキュメント: `docs/WCC_PRODUCT.md` セクション 5、`docs/WCC_BRAND.md`

---

## 背景

既存 LP は公開済みで、コピー・構成はおおむね完成している。
ただし以下の課題が残っている:

1. **SNS アイコンが空リンク** (Instagram/TikTok アカウント未設定) — サービス特性上、致命的
2. **海外事例がストック素材を疑われるリスク** (Lucy and Chris、Sophie & Daniel 等)
3. **問い合わせフォームが Supabase と未接続** — 受信できていない可能性

## 優先度高

### P0: SNS リンクの修正 ★最優先

現状 LP の SNS アイコンが `instagram.com/` トップに飛ぶ空リンクになっている。
作例が主な営業ツールであるこのサービスでは、SNS アカウントの不在は致命的。

- [ ] Instagram 実績アカウント開設後、LP の SNS リンクを更新
- [ ] TikTok 実績アカウント開設後、同様に更新
- [ ] `docs/INFRA_SETUP.md` の SNS アカウント項目と連動

### P1: 問い合わせフォームの Supabase 接続

現状の問い合わせ導線を `messages` テーブルと接続する。

- [ ] フォーム送信 → `messages` テーブルへ保存 (channel=form, direction=inbound)
- [ ] 運営者への Resend 通知メール
- [ ] 自動返信メール (簡易お礼文)

### P2: 実績セクションの信頼性向上

- [ ] 実際の納品実績を許諾取得の上で掲載 (海外ストック素材との差別化)
- [ ] Instagram / TikTok 埋め込みで実際の作例を表示

## 優先度中

### P3: メイン体験・ブランドへの整合

`docs/WCC_BRAND.md` のカラーパレット・タイポグラフィ・禁句リストとの整合確認。

- [ ] カラーパレットを BRAND.md の指定に合わせる (オフホワイト `#FAF8F5` / 墨灰 `#2A2A28` / 朝霧モーヴ `#C9A39A`)
- [ ] 禁句 (「奇跡の」「最高の」「AI 搭載」等) が使われていないか確認
- [ ] ヒーローコピーが「翌朝、二人で昨日を笑い合うための動画」の世界観と整合しているか確認
- [ ] 屋号「For Your Wedding Day」の表記ルール (BRAND.md セクション 1.3) を全ページに適用
- [ ] ヘッダーロゴが「WCC」表記になっている問題を修正（屋号 For Your Wedding Day と不一致）

### P4: 納品物閲覧ページへの導線整備

- [ ] パスワード保護の納品物閲覧ページを実装後、LP から分離した URL で提供
- [ ] 顧客向けのダウンロード案内文をブランドトーンに合わせて作成

## 優先度低 (Phase 2 以降)

- SEO 対策 (ターゲットキーワード選定、`#プレ花嫁` 検索流入の設計)
- 式場向けプランナー紹介資料ページ
- 実績ブログ・制作日誌セクション

## 変更凍結事項

以下は現時点では変更しない:

- **料金 (150,000 円税抜)** — 変更には `docs/WCC_PRICING_LOGIC.md` (未作成) の更新が必要
- **パッケージ内容** — 実撮影で確定前なので変更しない
- **全体的な LP 構成** — 大規模変更は Phase 2 以降で検討
