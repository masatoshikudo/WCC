# 結婚式当日撮影サービス（フォト・ムービー）事業者およびビジネスモデル調査レポート

本レポートでは、結婚式当日（挙式および披露宴会場）の撮影に特化したサービスを提供する国内外の事業者と、そのビジネスモデルについて、フォト（写真）とムービー（映像）に分けて整理・解説します。

---

## 1. 市場構造とビジネスモデルの概況

結婚式当日の撮影市場は、大きく分けて**「式場専属（インハウス・提携）モデル」**と**「外注（持ち込み）モデル」**の2つのビジネス構造が存在します。

*   **式場専属（提携）モデル**: 結婚式場が自社でカメラマンを雇用する、または特定の撮影会社と独占的（あるいは優先的）な提携を結ぶモデルです。新郎新婦にとっては手配が容易で、カメラマンが会場の動線やルールを熟知している安心感がありますが、式場側の中間マージン（仲介手数料）が上乗せされるため、料金は高額（写真・ビデオそれぞれ10万〜30万円程度）になる傾向があります [1]。
*   **外注（持ち込み）モデル**: 新郎新婦が外部の撮影業者やフリーランスのカメラマンを直接手配するモデルです。中間マージンが発生しないため、式場専属モデルと比較して半額から3分の1程度の費用（3万〜8万円程度）に抑えられることが最大のメリットです [2]。ただし、式場によっては「持ち込み料」を請求されたり、外部カメラマンの立ち入りエリア（メイク室や祭壇付近など）に制限が設けられたりするケースがあります。

近年では、これらに加えて、撮影データのオンライン販売プラットフォーム（SaaS）や、ゲストが撮影した素材を活用するクラウドソーシング型のビジネスモデルも登場しています。

## 2. 結婚式当日フォト（写真）撮影サービス

### 2.1 国内の主要事業者とビジネスモデル

国内の当日フォト撮影市場は、式場提携のBtoBtoCモデルで大規模に展開する企業と、価格競争力を武器にBtoCで直接受注する外注専門会社に二分されます。

| 企業名・サービス名 | ビジネスモデルの分類 | 特徴・強み |
| :--- | :--- | :--- |
| **ジャパンクリエイト株式会社** | 式場提携（BtoBtoC） | 全国80以上の結婚式場と提携し、社員フォトグラファーを派遣。撮影だけでなく、高品質なアルバム（キヤノン製プリンター導入等）の制作までワンストップで提供し、式場の単価アップに貢献するモデルです [3]。 |
| **株式会社フォトクリエイト（グロリアーレ）** | プラットフォーム（BtoB/SaaS） | 式場や撮影事業者向けに、ウエディングフォトのオンライン販売システム「グロリアーレ」を初期費用・システム利用料無料で提供。ゲストや親族への写真販売による売上の一部を手数料として得るレベニューシェアモデルです [4]。 |
| **株式会社カメオ** | 式場提携（BtoBtoC） | 首都圏を中心に、有名ホテルやゲストハウスと提携し、ブライダルスナップやスタジオ写真を提供する老舗企業です。式場のブランドに合わせた高品質な撮影を担保しています [5]。 |
| **Kando Wedding（カンドウ）** | 外注・持ち込み専門（BtoC） | 「結婚式持ち込み専門会社」として、スナップ写真を5万円（税別）という低価格で提供。式場との交渉アドバイスも行い、価格重視のカップルから大量受注するモデルです [6]。 |
| **PARTY PHOTO（パーティーフォト）** | 外注・持ち込み専門（BtoC） | 1日1組限定などのプランを用意し、式場専属カメラマンが撮影しない中座中やゲストのオフショットも積極的に撮影するホスピタリティを売りにしています [7]。 |

### 2.2 海外の主要事業者とビジネスモデル

海外（主に米国や欧州）では、日本のような「式場専属」の縛りが緩く、フリーランスのフォトグラファーと直接契約する文化が根付いていますが、近年は効率化を支援するプラットフォームが台頭しています。

| 企業名（国） | ビジネスモデルの分類 | 特徴・強み |
| :--- | :--- | :--- |
| **Complete Weddings + Events**（米国） | フランチャイズ展開 | 写真、ビデオ、DJ、フォトブースなどをパッケージ化し、全米でフランチャイズ展開する珍しいモデル。スケールメリットを活かした集客と品質の標準化を行っています [8]。 |
| **WedShoots**（米国・欧州等） | UGC（ユーザー生成コンテンツ）型 | The Knot / WeddingWireグループが提供する無料アプリ。ゲストがスマートフォンで撮影した当日の写真を一つのオンラインアルバムに集約できるプラットフォームです [9]。 |
| **Momento Pro**（豪州） | プロ向けアルバム制作（BtoB） | ウェディングフォトグラファー向けに、高品質なフォトブックやアルバムの制作・印刷を請け負うサービス。ソフトウェアを無償提供し、印刷代で収益を上げます [10]。 |

## 3. 結婚式当日ムービー（映像）撮影サービス

### 3.1 国内の主要事業者とビジネスモデル

ムービー領域では、単なる「記録ビデオ」だけでなく、披露宴の最後に上映する「撮って出しエンドロール（当日編集・ライブエンドロール）」の需要が非常に高く、これに特化した事業者が多く存在します。

| 企業名・サービス名 | ビジネスモデルの分類 | 特徴・強み |
| :--- | :--- | :--- |
| **株式会社映像商会** | 式場提携 / 外注請負 | 記録ビデオやダイジェスト編集を専門とする映像制作会社。高い技術力を持ち、式場からの一次請けと、個人からの直接発注の両方に対応しています [11]。 |
| **株式会社point zero** | 映像制作特化（BtoBtoC / BtoC） | ブライダルビデオを専門とし、シネマティックでドキュメンタリータッチの映像制作に定評があります [12]。 |
| **タニグチスタジオ** | 外注・持ち込み専門（BtoC） | 25年の実績を持つ外注専門業者。記録ビデオ撮影を4万円台〜という低価格で提供し、写真とのセット割引などで単価を確保するモデルです [13]。 |
| **ウェディングムービー制作 DMC** | 格安制作・外注専門（BtoC） | 撮って出しエンドロールや事前制作ムービーを格安で提供。撮って出しの場合は撮影者と編集者の2名体制が必要なため、単価は高くなりますが、外注専門ならではの価格競争力を持っています [14]。 |

### 3.2 海外の主要事業者とビジネスモデル

海外のムービー領域では、オンデマンド手配や、コロナ禍を経て定着したライブ配信、さらには編集のみを切り出したサービスなど、ビジネスモデルの細分化が進んでいます。

| 企業名（国） | ビジネスモデルの分類 | 特徴・強み |
| :--- | :--- | :--- |
| **Splento**（英国） | オンデマンドマッチング | 審査済みのプロビデオグラファーを時給制（£149/時〜）でオンデマンド手配できるプラットフォーム。撮影から48時間以内の納品を保証し、システム化による低価格とスピードを実現しています [15]。 |
| **Wedfuly**（米国） | ライブ配信特化型 | コロナ禍で急成長した、結婚式のオンラインライブ配信（バーチャルウェディング）に特化したサービス。機材の事前配送と、遠隔での配信ディレクション・サポートを提供するSaaS＋サービスモデルです [16]。 |
| **WeddingMix**（米国） | クラウドソーシング型 | ゲストに小型カメラ（フリップカム）を貸し出すか、専用アプリを使ってもらい、ゲストが撮影した動画素材をプロが編集して一つのウェディングムービーに仕上げる低価格モデルです [17]。 |
| **VidFlow / Viddrop**（米国） | 納品特化型プラットフォーム | ビデオグラファー向けに、完成した大容量の4Kウェディングムービーをクライアント（新郎新婦）に美しく納品・ストリーミング再生させるための専用プラットフォーム。サブスクリプションまたは買い切りで提供されます [18]。 |

## 4. 考察と今後のトレンド

結婚式当日の撮影サービス市場は、長らく「式場が提携業者を抱え込み、高単価で販売する」というクローズドな市場でしたが、SNSの普及やカップルの情報リテラシー向上により、**「高品質な外注業者を直接手配する（持ち込み）」**トレンドが国内外で加速しています。

特にムービー領域においては、以下のトレンドが顕著です。

1.  **即時性の重視**: 日本国内における「撮って出しエンドロール」の定着や、海外におけるSplentoのような「48時間以内納品」など、感動が冷めないうちのコンテンツ提供が価値を生んでいます。
2.  **UGCの活用とコンテンツクリエイターの台頭**: ゲストが撮影した写真・動画を集約するWedShootsやWeddingMixのようなサービスに加え、最近ではTikTokやInstagramのリール用に短い縦型動画を当日中に撮影・編集して納品する「ウェディング・コンテンツ・クリエイター」という新しい職業・サービスが海外を中心に急増しています [19]。
3.  **プロセスのアンバンドル化**: 撮影（カメラマン）、編集（エディター）、納品（プラットフォーム）の各工程が切り離され、それぞれに特化したSaaSやマッチングサービス（WedWillなどの編集専門外注、VidFlowなどの納品ツール）が普及し、業界全体の効率化が進んでいます。
