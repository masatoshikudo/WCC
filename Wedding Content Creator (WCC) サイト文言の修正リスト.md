# Wedding Content Creator (WCC) サイト文言の修正リスト

確定した8ステップのフロー（問い合わせ → ZOOM調整 → ZOOM → 見積り/決済リンク → 決済 → LINE誘導 → 当日までのやり取り → 挙式当日）と、現行のサイト（トップページ、予約フォーム、特定商取引法ページ）の文言を照合し、ユーザーに誤解を与える可能性がある箇所を洗い出しました。

## 1. トップページ（/）の修正点

### 該当箇所: パッケージプランの内容リスト
*   **現状の文言:** 「詳細について話し合うZOOM相談」「公式LINEチャット相談」
*   **誤解の要因:** このリストだけを見ると、ZOOM相談やLINEチャットが「150,000円を支払った後（契約後）のサービス」なのか、「契約前でも無料でできる」のかが曖昧です。確定フローでは、ZOOMは**契約前（見積もり前）**、LINEは**契約後**という明確な分かれ目があります。
*   **修正案:** パッケージ内容には「契約後の公式LINEチャット相談」と明記し、ZOOM相談については「※ご契約前にオンライン（ZOOM）での無料相談・お打ち合わせを実施します」と注記を追加することを推奨します。

### 該当箇所: FAQ「お支払いのタイミングは？」
*   **現状の文言:** 「ご相談内容の確認後に、見積と請求URLをメールでご案内します。お支払いは請求ページで一括（税抜）です。」
*   **誤解の要因:** 確定フローでは「ご相談内容の確認 → **ZOOMミーティング** → 見積と請求URLのご案内」という流れですが、現状の文言ではZOOMミーティングの存在が抜け落ちており、「フォームを送ったらすぐに見積もりが来て払わされる」という印象を与えかねません。
*   **修正案:** 「フォームよりお問い合わせいただいた後、オンライン（ZOOM）にて詳細をお伺いします。その後、最終的なお見積りと請求URLをメールでご案内します。お支払いは請求ページにて一括（税抜）となります。」

### 該当箇所: FAQ「予約後、日程はどのように確定しますか？」
*   **現状の文言:** 「ご相談フォーム送信後、担当確認と見積のご案内を経て日程を確定します。」
*   **誤解の要因:** ここでも「ZOOMミーティング」と「決済（お支払い）」のステップが省略されています。ユーザーにとって「見積の案内＝日程確定」なのか「支払い＝日程確定」なのかが不明確です。
*   **修正案:** 「ご相談フォーム送信後、ZOOMでのお打ち合わせを経てお見積りをご案内します。お見積り金額の決済（お支払い）が完了した時点で、正式に日程（ご予約）が確定となります。」

### 該当箇所: FAQ「最終Zoomでは何を確認しますか？」
*   **現状の文言:** 「当日の動線を確認します。」
*   **誤解の要因:** 「最終Zoom」という表現は、契約後・挙式直前に複数回のZOOMがあるような印象を与えます。確定フローではZOOMは「見積もり前（契約前）」の1回のみと想定されます。もし挙式直前にもZOOMを行うのであれば、フローに「直前ZOOM」を追加する必要があります。
*   **修正案:** （契約前の1回のみの場合）「お見積り前のZOOMお打ち合わせでは、ご希望の撮影内容や当日の動線、式場側のルールなどを確認させていただきます。」

### 該当箇所: FAQ「予約前に相談できますか？」
*   **現状の文言:** 「可能です。ご相談フォームまたはLINEから、日程と希望内容をご共有ください。」
*   **誤解の要因:** 確定フローでは「公式LINE誘導」は決済後（STEP 6）となっていますが、この文言では「予約前でもLINEから相談できる」と読めます。もし予約前はフォームのみの受付とするのであれば、矛盾が生じます。
*   **修正案:** （予約前はフォームのみの場合）「可能です。まずはご相談フォームより、日程（未定でも可）とご希望内容をお気軽にお知らせください。」

## 2. 予約フォームページ（/book）の修正点

### 該当箇所: 冒頭の説明文
*   **現状の文言:** 「1分で送信できます。内容確認後、担当より見積と請求URLをメールでご案内します。」
*   **誤解の要因:** トップページのFAQと同様に、フォーム送信後すぐに「見積と請求URL」が送られてくるような印象を与えます。確定フローである「ZOOMでのヒアリング」が抜けています。
*   **修正案:** 「1分で送信できます。内容確認後、担当より確認のメールをお送りし、オンライン（ZOOM）でのお打ち合わせ日程を調整させていただきます。」

## 3. 特定商取引法に基づく表記（/legal/tokushoho）の修正点

### 該当箇所: 支払時期
*   **現状の文言:** 「パッケージ料金は一括でお支払いいただきます（税抜価格）」
*   **誤解の要因:** 法律上求められる「時期（いつ支払うのか）」が記載されていません。「一括で」は支払方法・回数です。
*   **修正案:** 「ご予約確定時（ZOOMでのお打ち合わせ後、お見積りをご案内したタイミング）に一括でお支払いいただきます。」

---

## まとめと推奨アクション

現行サイトの文言は、全体的に「ZOOMミーティング（ヒアリング）」と「お支払い（決済）」のタイミングが省略されがちであり、これがユーザーの不安や誤解（いきなり請求されるのでは？いつ契約が成立するのか？）を招く原因となっています。

作成した「ご利用の流れ」セクションをサイトに追加すると同時に、上記のリストに挙げた箇所を修正することで、サイト全体で一貫した、ユーザーに安心感を与えるフローを提示できるようになります。
